Политика конфиденциальности

1. Общие положения

Настоящая Политика конфиденциальности определяет порядок обработки и защиты персональных данных пользователей сервиса Gorod Legends AI (далее — "Сервис").

2. Какие данные мы собираем

  • Анонимные идентификаторы пользователей Telegram
  • История сообщений для улучшения работы AI
  • Технические данные (IP-адрес, браузер, устройство)
  • Аналитические данные о взаимодействии

3. Цели обработки данных

  • Предоставление сервиса общения с AI собеседницами
  • Улучшение качества AI-ответов
  • Техническая поддержка пользователей
  • Анализ использования сервиса

4. Защита данных

Мы применяем современные методы шифрования и защиты данных. Все сообщения обрабатываются анонимно. Персональные данные не передаются третьим лицам без согласия пользователя.

5. Права пользователей

  • Право на удаление всех данных
  • Право на получение копии данных
  • Право на исправление данных
  • Право на отзыв согласия

6. Контакты

По вопросам конфиденциальности обращайтесь: privacy@gorod-legends.ru
